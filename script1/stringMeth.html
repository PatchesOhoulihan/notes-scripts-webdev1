Suchen und Finden mit indexOf

Mit indexOf können Sie die Position eines bestimmten Strings innerhalb eines größeren Strings herausfinden.

let name ="Ladislaus Jones"
console.log(name.indexOf(' '))
Rückgabewert = 9 (als position des Leerzeichens)

Mehr vom Index

  • Die Zählung beginnt von der Position 0
  • Wenn die Funktion indexOf den gewünschten Wert nicht findet, gibt sie -1 zurück.
  • indexOf gibt die Position des ERSTEN Fundes aus.
  • IndexOf beginnt am Anfang des Strings mit der Suche, Die Funktion lastIndexOf findet das letzte vorkommende Element.

Substring

Die Funktion substr extrahiert einen Teilstring aus einem größeren String. Als Argumente werden:
die Position des ersten Zeichen.
die Anzahl von Zeichen, die Sie extrahieren möchten.

let name="Ladislaus Jones"
name.substr(2,6); = rückgabewert "dislaus"

trim

trim entfernt Leerzeichen am Anfang und am Ende eines Strings, nicht jedoch in der mitte:
let name = ' Davy Jones '
name.trim() = 'Davy Jones'

toLowerCase und toUpperCase

Wandelt DEN KOMPLETTEN STRING entweder in groß oder kleinbuchstaben um.

"string".toUpperCase(); = "STRING"
"STRING".toLowerCase() = "string"

replace()

Mit der Funktion replace() ist es möglich innerhalb eines Strings Ersetzungen vorzunehmen. let story = "Die alte Knackwurst weiss nicht wie ihr geschieht. Das ist der Witz mit der alten Knackwurst"
let newStory = story.replace('Knackwurst', 'Salami');

NOTE: Dies ersetzt jedoch nur den ersten teilstring mit dem neuen Wert. Wenn ALLE erstezt werden sollen, dann muss Regular Expression als ersten Parameter verwendet werden.
/suchbegriff/g
Die Schrägstriche / signalisieren die Regulären Ausdrücke, das g steht für global und sorgt dafür das die Textersetzung für den ganzen Text durchgeführt wird
story.replace(/knackwurst/g, "Salami") (NOTE: reguläre Ausdrücke werden nicht in Anführungszeichen gesetzt.)

Funktion Zweck Beispielaufruf für let s = "abcde" Rückgabewert
charAt ermittelt das Zeichen an der angegebenen Stelle s.charAt(1) b
includes Prüft, ob der String den angegebenen Teilstring enthält s.includes('bc') true
startsWith Prüft, ob der String mit angegebenen Teilstring beginnt. s.startsWith('ab') true
endsWith Prüft, ob der String mit dem angegebenen Teilstring endet .endsWith('de') true
indexOf Ermittelt die Position des angegebenen Teilstrings innerhalb des Strings s.indexOf('cd') 2
lastIndexOf Ermittelt die Position des angegebenen Teilstrings innerhalb des Strings(Suche beginnt von hinten) s.lastIndexOf("de") 3
repeat Wiederholt den String so oft wie angegeben s.repeat(2) abcdabcdabcd
substr Extrahiert einen Teilstring an der angegebenen Postion s.substr(1,3) bcd
toLowerCase Gibt den String in Kleinschreibung zurück. "ABcdE".toLowerCase() abcde
toUpperCase Gibt den String in Großschreibung zurück "AbCdE".toUpperCase() ABCDE
trim Entfernt Leerzeichen am Anfang und am Ende " abc ".trim() abc

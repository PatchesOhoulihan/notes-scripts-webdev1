Was Sie schon immer über Javascript wissen wollten

Javascript?

Browser können neben JavaScript auch andere Sprachen ausführen, z. B. Dart (in Google Chrome) oder VBScript (im Microsoft Internet Explorer). JavaScript ist aber die einzige Pro- grammiersprache, die alle modernen Browser zuverlässig unterstützen.

Browser sind auch längst nicht die einzige Plattform, die JavaScript ausführen können. So finden Sie Java- Script beispielsweise auch innerhalb von PDFs, zum Scripten von Anwendungen wie Photoshop oder in der Steuerungssoftware von Robotern 5

JavaScript auf dem Server auszuführen ist keine neue Idee. Eine ernstzunehmende Ver- breitung hat die Sprache im Serverbereich aber erst seit ca. 2010/2011 erreicht, im Wesentli- chen durch die Plattform Node.js 6 .

Schließlich liegen mit Espruino 7 und Tessel 8 zwei Boards vor, die JavaScript direkt auf der Hard- ware ausführen.(embedded)

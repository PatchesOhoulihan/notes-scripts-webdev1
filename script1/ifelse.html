5 Sehr bezeichnend, diese Bezeichner

Besonderheiten bei nur einer Anweisung

Wenn Sie übrigens im if - oder im else -Zweig nur eine einzige Anweisung haben, können Sie auf die geschweiften Klammern verzichten. Wir empfehlen aber, die Klammern nur dann wegzulassen, wenn es sich um eine kurze Anweisung handelt, die Sie mit dem if in die gleiche Zeile schreiben können.

if (answer === SOLUTION) console.log("42 is correct.");

Der tertiäre Operator


                        if (productCategory === "books") {
                            vatRate = 1.07;
                        } else {
                            vatRate = 1.19;
                        }

Dass Sie in beiden Zweigen einer if-Anweisung derselben Variable einen Wert zuweisen, kommt recht häufig vor. Wenn die if-Anweisung, so wie hier, keinen weiteren Zweck hat, kann der sogenannte tertiäre Operator helfen, die Lesbarkeit des Codes zu verbessern.

Der tertiäre Operator hat drei Operanden (deswegen tertiär) und trennt diese mit den Zeichen ? und : . Sie können den obigen Code auch so schreiben

vatRate = productCategory === "books" ? 1.07 : 1.19;

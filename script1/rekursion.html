5 Sehr bezeichnend, diese Bezeichner

So einfach es auch ist, dass sich eine Funktion selbst aufrufen kann, erscheint es doch zunächst ein wenig gewöhnungsbedürftig, oder? Genau das können Funktionen. Programmierer, die halbwegs gebildet sind, bezeichnen das Prinzip übrigens als Rekursion.

Wenn Sie hier einen Fehler machen, ist es möglich, dass das Programm in eine Endlos- schleife gerät und dadurch »hängenbleibt«. Sie müssen daher auf jeden Fall dafür sorgen, dass die Abbruchbedingung auch irgendwann erfüllt wird.

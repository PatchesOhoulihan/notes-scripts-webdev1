Das Math Objekt

Sie können damit Werte runden, Zufallszahlen bestimmen oder Winkelberechnungen durchführen. Damit all diese Funktionen nicht sinnlos in JS verstreut sind, gibt es das Math -Objekt.

Function oder Konstante Zweck Beispiel
sqrt Quardratwurzel Math.sqrt(9) ergibt 3
pow Potenzieren Math.pow(2, 5) ergibt 32
cos Kosinus Math.cos(0) ergibt 1
PI PI Math.PI ergibt 3.141592653589793

Eine runde Sache: floor, round & ceil

Auch die häufig genutzten Rundungsfunktionen verbergen sich hinter dem Math-Objekt. Es gibt drei Rundungsfunktionen: round , ceil und floor . Die Funktion round ist ein Klas- siker. Sie verhält sich so, wie Sie es aus dem Geschäftsleben kennen: Ab einem Wert von 0.5 nach dem Dezimalpunkt wird aufgerundet. Die Funktion floor rundet dagegen immer ab und ceil immer auf. Beispiel:
➤ Math.round(3.1) ergibt 3
➤ Math.round(3.5) ergibt 4
➤ Math.floor(3.7) ergibt 3
➤ Math.ceil(3.1) ergibt 4

Frisch gefixt mit toFixed

Manchmal möchten Sie Zahlen auf eine bestimmte Anzahl von Nachkommastellen runden. Beispielsweise ist es üblich, Geldbeträge auf zwei Stellen zu runden. Diese Möglichkeit finden Sie allerdings nicht — wie Sie vielleicht vermuten würden — bei Math . Stattdessen rufen Sie die Funktion toFixed direkt auf einem Zahlenwert auf.
3.4567.toFixed(3); // => 3.457

Zufallszahlen mit random()

Die Funktion Math.random() liefert Zahlen zwischen 0 und 1, wobei die 1 nie erreicht wird.

Funktion oder Konstante Zweck Beispiel
sqrt Quadratwurzel Math.sqrt(9) = 3
pow Potenzieren Math.pow(2,5) = 32
cos Kosinus Math.cos(0) = 1
PI pi Math.PI = 3.141592653589793
round kaufmännisches Runden Math.round(3.456) = 3
floor Abrunden Math.ceil(3.678) = 3
ceil Aufrunden Math.ceil(3.678) = 4
max Maximalwert aus beliebig vielen Argumenten Math.max(3,7,2) = 7
min Minimalwert aus beliebig vielen Argumenten Math.min(3,7,2) = 2
abs Betrag einer Zahl ohne Vorzeichen Math.abs(-3.678) = 3.678
random Eine zufällige Zahl Math.random() = eine Zahl zwischen 0 und 1
toFixed Runden auf beliebig viele Nachkommastellen 3.0.toFixed(2) = 3.00

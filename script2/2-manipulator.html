Hello DOM mit querySelector und innerHTML

HTML:
<h1>Hello <em>World</em></h1>
<p>Lorem ipsum dolor sit amet.</p>

JS: 

                        
                            document.querySelector('h1').innerHTML = "Hello Dom";

Ein Blick hinter die Kulissen

Mit der Funktion document.querySelector können Sie HTML-Elemente innerhalb des HTML-Dokuments auswählen. Der Parameter ist ein beliebiger CSS-Selector
document.querySelector('h1') returns First h1-Element as JS-Object

Name Beispiel Beschreibung
Univeral selector * Selektiert ein beliebiges Element. Diesen Selektor sollten Sie nicht ohne weitere Einschränkungen verwenden, da das Selektieren aller Elemente einer Website sehr viel Rechenzeit in Anspruch nehmen kann.
Type Selector h1 Selektiert alle Elemente eines bestimmten Typs
ID Selector #some-id Selektiert genau ein Element mit der gegebenen id
Class selector .some-class Selektiert alle Elemente, denen die angegebenen Klasse zugewiesen wurde. Beachten Sie, dass ein Element mehere Klassen haben kann.
, h1, p, .go Das Komma ist selbst kein Selektor, sondern erlaubt es, mehrere Selektoren zu kombinieren. Im Beispiel würden alle h1-Elemente, alle p-element und alle Elemente der Klasse go selektiert.

Genauer gesagt handelt es sich bei dem Rückgabewert von document.querySelector um ein Element-Objekt

Der Rückgabewert von .innerHTML ist ein String. Der Wert von innerHTML lässt sich mit .innerHTML = irgendwas ändern

Suchen und finden von document.querySelectorAll im DOM

Mit Hilfe von document.querySelector lassen sich JS Element Objecte aus dem DOM extrahieren. Tatsächlich ist das ganze HTML-Dokument nichts anders als eine Sammelung von HTML-Elementen, die in einer Baumstruktur angeordnet sind. Dem sogenannten DOM(Document Object model)

..sie alle zu finden ... - querySelectorAll

Im Gegensatz zu querySelector, das den ersten Treffer zurückgibt, werden mit querySelectorAll alle zum Selektor passenden Elemente zurück gegeben.

Als Rückgabewert erhält man eine NodeList das mehrere Elemente(Konten/Nodes) in einer Liste zusammenfasst

Die Nodeliste verhält sich ähnlich wie ein Array. Sie können mit dem Index-Operator auf seine Elemente zugreifen:
document.querySelectorAll('h2')[0].innerHTML

document.querySelectorAll('h2').length lässt sich die Menge der gefunden Elemente überprüfen.

Um die Elemente einer Nodelist durchlaufen zu können muss eine forEach Schleife verwendet werden. Vorher muss die Nodeliste jedoch noch in ein Array umgewandelt werden.
Example: 

                
                    let heading = Array.from(document.querySelectorAll('h2'));
                    heading.forEach(h => console.log(h.innerHTML));

Attribute Selektoren

Mit diesen ist es möglich auf Elemente mit bestimmten Attributen und Werten zuzugreifen.

Selektoren Beispiel zu
<img src="funny_cat.png"/>		 Beschreibung
[attr] [src] Selektiert alle Elemente mit dem angebenen Attribut. Der Attributwert spielt dabei keine Rolle(attribute presence)
Example: document.querySelector('[src]');
[attr=val] [src="funny_cat.png"] Selektiert Elemente, bei denen das angebene Attribut mit GENAU dem angebenen Wert(funny_cat.png) enthalten(exact attribute match)
Example: document.querySelector("[src='funny_cat.png']")
[attr*=val] [src*=cat] Selektiert Elemente, bei denen das angebene Attribute(src) den Wert(cat) als Teilstring enthält (substring attibute match)
Example: document.querySelector('[src*=cat]')
[attr^=val] [src^=funny] Selektiert Elemente, bei denen das angegebene Attribute(src) mit dem Wert(funny) beginnt.
Example:
[attr$=val] [src$=png] Selektiert Elemente, bei denen das angegebene Attribut(src) auf den angegebenen Wert(png) endet.

Combinator Selektoren

Name Beispiel Beschreibung
Descendent Combinator(Leerzeichen) li a Selektiert Elemente, die die Nachfahren eines anderen angegebenen Elements sind. Im Beispiel werden alle a-Elemente selektiert, die Nachfahren von li sind, egal wie viele Ebenen dazwischen liegen.
Next Sibling Combinator (+) li#start + li Mit diesem Selektor können Sie das Element auswählen, das im HTML-Quelltext DIREKT NACH dem angegeben Element auf der gleichen Ebene(Geschwister Element) folgt.
General Sibling Combinator li#start ~ li Mit diesem Selektor können Sie alle Elemente auswählen, die im HTML-Quelltext nach dem angegeben Element auf der gleichen Ebene (Geschwister-Element) folgen. Es wird NICHT NUR DAS DIREKT FOLGENDE Element gewählt, wie beim Next sibling combinator.
Child Selector li > a > span Mit diesem Selektor lassen sich Elemente auswählen die DIREKTE NACHFAHREN von einander sind.

Pseudo Classes

Mit sogenannten pseudo classes erlauben es, Selektoren weiter zu spezialisieren.

Name Beispiel Beschreibung
:first-child ul li:first-child Mit dieser Pseudo-Klasse sprechen Sie das erste Kind-Element eines übergeordneten Containers(ul) an. Sollte das Element(li) NICHT den richtigen Type haben wird nichts selektiert.
:last-child ul li:last-child Aquivalent zu :last-child, selektiert das letzte Element.
:first-of-type div p:first-of-type Mit dieser Pseudo-Klasse sprechen Sie immer das erste Kind eines angegebenen Element-typs an.
:last-of-type div p:last-of-type Äquivalent zu :first-of-type, spricht immer das letzte Element an.
:only-child #article p:only-child Diese Klasse selektiert ein Element nur, wenn es sich um das EINZIGE KIND handelt.
:empty div:empty Spricht nur Elemente an, die ohne Kind-Elemente sind. Inhalt in Form von Text wird dabei ebenfalls als Kind-element betrachtet, Kommentare jedoch nicht.
:nth-child(x) ul li:nth-child(2) Spricht das n-te Element an
:nth-type-of-child(x) div p:nth-type-of-child(5) Spricht das n-te element vom angegeben typ an.
:not(selector) :not(span) Negation. Wählt Elemente aus, wenn sie nicht dem in Klammern angegeben Selektor entsprechen. Im Beispiel würden alle Elemente bis auf das span angesprochen werden.

Kurz und Knapp

Da man document.querySelectorAll in der Praxis sehr häufig tippen muss wäre eine kürzere schreibweise extrem nütztlich.

                
                    const $ = document.querySelector.bind(document);
                    const $$ = document.querySelectorAll.bind(document);

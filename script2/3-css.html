classList und DOMTokenList

Jedes NodeObject hat neben der Eigenschaft innerHTML auch eine Eigenschaft namens classlist, das seine CSS-Klassen repräsentiert

document.querySelector('#diesdas').classList

Sie erhalten eine sogenannte DOMTokenList zurück, diese verhält sich ähnlich wie ein Array. Sie können zB. die Eigenschaft .length verwenden. In der DOMTokenList können Elemente mit der dem gleichen Wert NICHT zweimal vorkommen. Auch die Methoden verhalten sich unterschiedlich und zwar gibt es KEIN push, pop ,sort

Methoden von classList

Mehr Klassen mit add

document.querySelectorAll('li').classList.add('highlight')

Was zuviel ist, ist zuviel - mit remove

document.querySelectorAll('li').classList.remove('highlight')

Wechselspiel mit toggle

document.querySelectorAll('li')classList.toggle('highlight')

Are you there - mit contains

document.querySelectorAll('li').classList.contains('highlight')

Nur highlighten, was man highlighten will - mit filter

let foundNodes = (Array.from(document.querySelectorAll('#chat_members'))).filter((liNode => {liNode.innerHTML.includes(searchFor)}))
let highligtSelectedNodes = foundNodes.forEach(li => li.classList.add('highlighted'))

Methoden sind eine spezielle Art von Funktionen. sie unterscheiden sich von anderen Funktionen dadurch, das Sie zu einem Objekt gehören.

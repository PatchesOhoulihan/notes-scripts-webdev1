Events im Browser

Für den Browser ist schon jede kleine Einflussnahme durch den Anwender ein Event

  • das Anklinken eines Buttons
  • das Überfahren eines Bildes mit des Maus
  • das Loslassen einer Taste
  • das Verlassen eines Eingabefeldes
  • Fingergesten auf Touchbildschirmen, wie z.B Zoom

Es gibt auch Events, die nicht direkt vom Anwender ausgelöst werden, zB durch Netzwerkanfragen oder bei abgeschlossenen Ladevorgängen von Dateien

Ein Event erstellen

hierfür benötigt man die Methode addEventlistener Nahezu jedes HTML-Element verfügt über diese Methode.

Example: document.querySelector('#member').addEventListener('keyup', function(event){})

Die Function besitzt 2 Argumente:

  • Das erste Argument ist der Eventtyp
  • Das zweite Argument ist eine Function die der Browser ausführt wenn das Event auftritt.

Guard clauses: Wächter für ihre Functionen

if(partOfmemberName === "") Zeilen wie diese, die vor der Ausführung einer Function prüfen, ob die eingehenden Argumente einer Function überhaupt Sinn machen heissen Guards-clauses oder einfach Guards. Sie schützen functionen vor ungültigen Eingabewerten.

Schauen Sie hinter die Fassade - mit dem Event-Object

Wenn mit .addEventListener('event' function(event){}) sich auf ein bestimmtes Element registriert. Wird das event-object als Argument and an die Callback function übergeben.

Example: document.querySelector('input').addEventListener('keyup' function(event){console.log(event);}), so kann man sich den Inhalt des Eventobject ausgeben lassen

Eigenschaft Inhalt

65 65
timestamp Timestamp, wann ein Event sich ereignet hat.

Es gibt noch viel mehr Eigenschaften, doch schon diese wenigen verdeutlichen, dass Sie Eventobjekten alle Informationen Finden, die beim Auftreten des Events eine Rolle spielen

preventDefault oder: Wer mag schon immer nur "standard"

um das standardverhalten eines Elements zu verhindern kann aufdem Eventobject die preventDefault() Methode aufgerufen werden. Example:
document.querySelector('input').addEventListener('keyup' function(event){event.preventDefault();})
